Secure Sockets Layer (SSL)

A protocol from Netscape Communications Corporation designed to provide secure communications on the Internet. SSL works by using a private key to encrypt data transferred over the SSL connection. Many Web sites use the protocol to obtain confidential user information, such as credit card numbers. By convention, Web pages that require an SSL connection start with https: instead of http:.
