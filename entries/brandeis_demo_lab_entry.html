Brandeis DEMO Lab

DEMO (Dynamic and Evolutionary Machine Organization) attacks problems in agent cognition using complex machine organizations that are created from simple components with minimal human design effort. DEMO studies recurrent neural networks, evolutionary computation, dynamical systems as substrates, and builds working systems to test theories.
