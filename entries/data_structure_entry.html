Data Structure

A specification and an implementation of a collection of one or more items of data, as well as the operations necessary to interact with the collection. The specification is the definition of the data structure as an abstract data type. The specification forms the programming interface for the data structure. The implementation is a concrete data type expressed in a programming language.
